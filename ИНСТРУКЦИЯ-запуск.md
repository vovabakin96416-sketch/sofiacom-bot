# 🤖 Инструкция: запуск бота Софии

## Что умеет бот
- Ловит слова **КАРТА / КОФЕ / РУНА** в комментариях (с опечатками и любым регистром)
- Отвечает **случайным текстом** из пула прямо в комментарии
- Тексты редактируются **прямо в Telegram** — GitHub и код не нужны
- Один человек получает ответ раз в 24 часа

---

## Шаг 1 — Узнай свой Telegram ID

Напиши боту @userinfobot → он пришлёт твой ID (число вида `123456789`).
Это нужно чтобы только ты могла управлять текстами.

---

## Шаг 2 — Создай бота в Telegram

1. Открой @BotFather → `/newbot`
2. Задай имя и @username
3. Скопируй **токен**: `7123456789:AAFxxxxxxxxxxxxxxx`
4. Отправь BotFather: `/setprivacy` → выбери бота → `Disable`
   *(без этого бот не увидит сообщения в группе)*

---

## Шаг 3 — Добавь бота в группу-обсуждение канала

1. Настройки канала → «Обсуждение» → подключи группу
2. Добавь своего бота в группу как **администратора**
3. Права: ✅ Читать сообщения, ✅ Отправлять сообщения

---

## Шаг 4 — Деплой на Koyeb (бесплатно навсегда 🎉)

Koyeb — реально бесплатный хостинг без срока действия. Карта не нужна.

### 4.1 — Загрузи файлы на GitHub

1. Зайди на **github.com** → New repository → название `sofia-bot` → Create
2. Загрузи три файла (кнопка «Add file → Upload files»):
   - `bot.py`
   - `requirements.txt`
   - `Procfile`

### 4.2 — Подключи Koyeb

1. Зайди на **koyeb.com** → Sign up (через GitHub)
2. «Create App» → «GitHub» → выбери репозиторий `sofia-bot`
3. **Build command**: оставь пустым
4. **Run command**: `python bot.py`
5. **Instance type**: выбери **Free**

### 4.3 — Добавь переменные окружения

В настройках приложения Koyeb → вкладка **Environment variables**:

| Name | Value |
|------|-------|
| `BOT_TOKEN` | токен от BotFather |
| `ADMIN_ID` | твой Telegram ID (из шага 1) |

Сохрани → Koyeb запустит бота.

### 4.4 — Проверь

- В логах Koyeb должна быть строка `Бот запущен ✅`
- Напиши в группе слово `карта` — бот ответит

## Как управлять текстами прямо из Telegram

Напиши боту в **личку** команду `/menu` — откроется панель с кнопками.
(Старые команды `/list`, `/addkarta`, `/delete` больше не нужны — всё через кнопки.)

```
/menu
```

Дальше:
1. **💬 Триггеры в комментах** — тексты на слова карта / кофе / руна.
   **💫 Кнопки на постах** — тексты для кнопок любовь / деньги / карты.
2. Выбери слово или кнопку → увидишь, сколько текстов в пуле.
3. Кнопки внутри:
   - **📋 Показать все** — список с номерами
   - **➕ Добавить предсказание** — пришли текст; `{name}` подставит имя подписчика
   - **🗑 Удалить предсказание** — пришли номер из списка
4. `/cancel` — выйти из режима ввода.

### Другие команды (для тебя как админа)
| Команда | Что делает |
|---|---|
| `/status` | Что сейчас: неделя, пост на сегодня, каналы |
| `/schedule` | План постов на текущую неделю |
| `/preview` | Прислать сегодняшний пост с кнопками одобрения |
| `/post` | Сразу опубликовать сегодняшний пост в канал |
| `/test` | Прислать сегодняшний пост в ТЕСТ-канал |
| `/testpost <id>` | Прислать конкретный пост по номеру в тест-канал |
